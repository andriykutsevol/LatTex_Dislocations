\section{Базисные векторы, и формы.}

\subsection{Базисные векторы.}

Вернемся к формуле (арнольд $ L_{v}f $). Пусть $ t $ произвольный пареметр. 
%шутц стр 56.
Это равенство справедливо для любой функции $ f $, так что мы можем записать
\beq \label{oper_diff} % 2.3
\frac{d}{dt} = \sum_{i} \frac{d^{i}}{dt} 
\frac{\partial}{ \partial x^{i} }.
\eeq


Набор чисел $ d x^{i}/dt $ можно считать компонентами вектора, касательного к кривой $ x^{i} (t) $

Каждая кривая имеет единственный касательный вектор. Разумеется, каждый вектор является касательным к бесконечному числу кривых, проходящих через точку $ x $ и для этого имеются  две различные причины. Во-первых, сущствует много кривых, касающихся друг друга и имеющих одинаковый касательный вектор в точке $ x $; во-вторых, данный путь (труекторию) можно перепараметризовать так, чтобы касательный вектор в $ x $ не изменился. Это изображено на рис.

В качестве простого примера рассмотрим кривую $ x^{i}(t) = ta^{i} $, где числа $ \lbrace a^{i} \rbrace $ -  константы. Тогда если $ x $ - точка $ t=0 $, то касательный вектор имеет вид $ dx^{i}/dt = a^{i} $. Другая кривая $ x^{i}(\mu) = \mu^{2}b^{i} + \mu a^{i} $ также проходит через $ x $ при $ \mu = 0 $ и имеет такой же касательный вектор в этой точке, $ dx^{i}/d\mu = a^{i} $. Перепараметризация кривой $ x^{i} = ( \mu^{3} + \mu )a^{i} $ дает кривую проходящую через те же самые точки и имеющую в точке
$ x (\mu = 0) $ тот же самый вектор характеризующий целый класс эквивалентности кривых, проходящих через данную точку.

Пусть $ a $ и $ b $ - произвольные два числа и $ x^{i} = x^{i}(\mu) $ - другая кривая, проходящая через точку $ x $. Тогда в этой точке будем иметь
\beq
\frac{d}{d\mu} = \sum_{i}
\frac{d x^{i} }{d \mu } =
\sum^{i} \frac{d x^{i} }{d \mu }
\frac{ \partial }{ \partial x^{i} }
\eeq
и
\beq
a \frac{d}{dt} + b \frac{d}{d \mu} = \sum_{i}
\left(
a \frac{d x^{i} }{ dt } + b \frac{d x^{i}}{\mu}
\right)
\frac{\partial}{ \partial x^{i} }
\eeq
Итак, числа $ \lbrace adx^{i}/dt + bdx^{i}/d\mu \rbrace $ являются компонентами нового вектора, который, конечно же, касается \textit{некоторой} кривой, проходящей через точку
$ x $. Следовательно, должна существовать кривая с параметром, скажем $ \lambda $, такая что в точке $ x $
\beq
\frac{d}{d \lambda} = \sum_{i}
\left(
a \frac{d x^{i} }{ dt } + b \frac{d x^{i}}{\mu}
\right)
\frac{\partial}{ \partial x^{i} }
\eeq
Сопоставляя полученные результаты, заключаем, что в точке
$ x $
\beq
a \frac{d}{dt} + b \frac{d}{ d \mu } = \frac{d}{ d \lambda }
\eeq
Следовательно, операторы дифференцирования вдоль кривых
( типа $ d/dt $ ) образуют \textit{векторное пространство} в точке $ x $. В каждой системе координат имеются специальные кривые, а именно сами координатные линии. Ясно, что операторы дифференцирования вдоль этих кривых просто $ \partial / \partial x^{i} $ и соотношение (2.3) показывает, что каждый оператор $ d / dt $ может быть представлен в виде линейной комбинации частных производных
$ \partial / \partial x^{i} $. Поэтому система $ \lbrace \partial / \partial x^{i} \rbrace $ образует базис указанного векторного пространства. Из равенства (2.3) вытекает, что вектор $ d / dt $ имеет в этом базисе компоненты $ \lbrace dx^{i} / dt \rbrace $. Мы и приходм в результате к тому, что простарнство всех касательных векторов в точке $ x $ и пространство дифференцирований вдоль кривых, проходящих через $ x $, находится во взаимно-однозначном соответствии.

В определении $ d/dt $ как касательного вектора к кривой
$ \varphi $ не используется перемещения на конечные расстояния, в нем нет упоминания о координатах; в частности, не используются понятия типа "преобразуется так же, как...", и используется производная как своего рода "движение" вдоль кривой, порождаемой касательным вектором: такое соединение понятийя анализа - производной с понятим геометрии - вектором имеет далеко идущие последствия.

Нужно, однако, иметь в виду, что складывать можно лишь векторы, приложенные к одной точке. Векторы приложенные к разным точкам, не имеют никакого отношения друг к другу. Касательные векторы лежат не в $ M $ (рис со сферой), а в касательном пространстве к $ M $ в точке $ x $, которое обозначается через $ L_{x} $.

Для любой точки $ x $ простарнство $ L_{x} $ является векторным пространством той же размерности $ n $, что и само многообразие. Любая совокупность $ n $ линейно-независимых векторов в $ L_{x} $ образует \textit{базис} в $ L_{x} $. Выбирая тот или иной базис в каждом $ L_{x} $ для всех точек $ x $ из $ M $, мы получает базисные векторные поля. Если в корестности $ U $ точки $ x $ задана система координат $ \lbrace x^{i} \rbrace $, то в каждой точке из $ U $ определен \textit{координатный бзис} $ \lbrace \partial/ \partial x^{i} \rbrace $.

Но вовсе не обязательно работать с координатными базисами - векторы можно записывать и по отношению к произволному базису $ \lbrace \bar{e}_{i} \rbrace $. Здесь индекс $ i $ используется для нумерации базисных векторов. Он не обозначает компоненту чего либо. В точке $ x $ произвольный вектор $ \xi $ может быть записат в виде
\beq
\xi = \sum \xi^{i}\frac{ \partial }{\partial x^{i}} =
\sum_{j} \xi^{j'}\bar{e}_{j}.
\eeq



Числа $ \lbrace \xi^{i} \rbrace  $ являются компонентами ветора $ \xi $ относительно базиса $ \partial/ \partial x^{i} $. Числа $ \lbrace \xi^{j'} \rbrace $ - компоненты $ \xi $ относительно $ \lbrace \bar{e_{j}} \rbrace $; они связаны с $ \xi^{i} $ согласно обычному закону преобразования векторов.

Если $ \xi $ и базисы $ \lbrace \partial/ \partial x^{i} \rbrace $ и $ \lbrace \bar{e_{j}} \rbrace $ рассматриваются как векторные поля, то компоненты $ \lbrace \xi^{i} \rbrace $ и $ \lbrace \xi^{j'} \rbrace $ поля $ \xi $ являются функциями на
$ M $. Векторное поле называется дифференцируемым, если эти функции дифференцируемы.

Условие существования системы линейно-независимых векторов
$\partial/ \partial x^{i} $ для произволной координатной системы в каждой точке $ x $ из $ U $ есть в точности условие того, что координаты являются \textit{хоршими} в точке $ x $, т.е. дают 1-1-отображение некоторой окрестности $ U $ точки $ x $ на соответствующую область $ V $ в $ R^{n} $.

Рассмотрим какую-нибудь хорошую систему координат на $ U $, скажем $ \lbrace y^{i}, i = 1,...,n \rbrace $. Тогда отображение из $ \lbrace ( x^{1},...,x^{n} ) \rbrace $ в $ U $ можент быть записано в виде
\beq
y^{j} = y^{j}( x^{i},...,x^{n} ), \hspace{1ex}.
\eeq

По теореме об обратной функции это отображение взаимно-однзначно, если и только если матрица Якоби $ \partial y^{i}/ \partial x^{i} $ невырождена.
Это означает, что в каждой точке из $ U $ векторы с компонентами
\beq
( \partial y^{1}/ \partial x^{1}, \partial y^{2}/ \partial x^{1},..., \partial y^{n}/ \partial x^{1} )\\
( \partial y^{1}/ \partial x^{2}, \partial y^{2}/ \partial x^{2},..., \partial y^{n}/ \partial x^{2} )\\
...\\
( \partial y^{1}/ \partial x^{n}, \partial y^{2}/ \partial x^{n},..., \partial y^{n}/ \partial x^{n} )
\eeq
линейно-независимы. Но это как раз и есть компоненты векторов
$ \lbrace \partial / \partial x^{i}, i = 1,...,n \rbrace $ относительно координатного базиса для системы координат $ \lbrace y^{i} \rbrace $, поскольку, согласно цепному правилу,
\beq
\frac{ \partial }{ \partial x^{1} } = 
\frac{ \partial y^{1} }{ \partial x^{1} }
\frac{ \partial }{ \partial y^{1} } + 
\frac{ \partial y^{2} }{ \partial x^{1} }
\frac{ \partial }{ \partial y^{2} } +
...+
\frac{ \partial y^{n} }{ \partial x^{1} }
\frac{ \partial }{ \partial y^{n} }
\eeq
и аналогично для остальных $ x^{i} $.%\ref{}

Итак, действительно, система $ \lbrace x^{i} \rbrace $ координат является хорошей, если и только если $ \lbrace  \partial/ \partial x^{i} \rbrace $ образуют базис пространства касательных векторов для каждой точки из $ U $.

\subsection{Базисные 1-формы. }

Отображение которое сопоставляет векторау вещественнозначное число мы определили как 1-форму. В связи с этим 1-форма являетя функцией на векторах.

Рассмотрим касательное пространство $ L_{x} $. Определим на $ L_{x} $ 1-форму $ \omega $, и рассмотрим множество таких 1-форм. Обозначим его $ L_{x}^{\ast} $.

Определим опрецию сложения для двух элементнов множества $ L_{x}^{\ast} $
\beq
( \omega + \sigma )( \xi ) = \omega ( \xi ) + \sigma ( \xi )
\eeq
и операцию умножения на число:
\beq
(\lambda \omega ) ( \xi ) = \omega ( \xi ) + \sigma (\xi).
\eeq

Эти две операции обеспечивают свойство которое называется линейнностью
\beq
\omega( \alpha \xi_{1} + \beta \xi_{2} ) = 
\alpha \omega (\xi_{1}) + \beta \omega (\xi_{2}).
\eeq
где $ \alpha $ и $ \beta $ - вещественные числа.
Таким образом мы определили линейную структуру на множестве
 $ L_{x}^{\ast} $, это занчит, что теперь совокупность 1-форм в данной точке образует линейное (векторное) простарнство.

Пространства $ L_{x}^{\ast} $ и $ L_{x} $ называются \textit{сопряженными} другу если на $ L_{x}^{\ast} \times L_{x} $ задана \textit{свертка}.

Функция определенная на $ L_{x}^{\ast} \times L_{x} $ называется сверткой, если соблюдены следующие условия:\\

1) Линейность по первому аргументу:
%%
$$
( \alpha \omega_{1} + \beta \omega_{2}, \xi ) = 
\alpha(\omega_{1}, \xi)  + \beta( \omega_{2}, \xi )
$$
для любых $ \alpha, \beta \in \mathbb{R} $, $ \omega_{1}, \omega_{2} \in L_{x}^{\ast} $, $ \xi \in L_{x} $( $ \mathbb{•} $, как обычно обозначает множество действительных чисел ).

2) Линейность по второму аргументу:
$$
( \omega, \alpha \xi_{1} + \beta \xi_{2} ) = 
\alpha(\omega, \xi_{1}) + \beta(\omega, \xi_{2})
$$
для любых $ \alpha, \beta \in \mathbb{R} $, $ \omega \in L_{x}^{\ast} $, $ \xi_{1}, \xi_{2} \in L_{x} $.

3) Невырожденность по первому аргументу: если $ ( \omega, \xi ) = 0 $ при данном $ \omega $ и при любом $ \xi \in L_{x} $, то
$ \omega = \theta^{\ast} $ (где $ \theta^{\ast} $ - нулевой элемент в $ L_{x}^{\ast} $).

4) Невырожденность по второму аргументу: если $ ( \omega, \xi ) = 0 $ при любом $ \omega \ in L_{x}^{\ast} $ и при данном $ \xi $, то $ \xi = \theta $ (где $ \theta $ - нулевой элемент
в $ L_{x} $).


Предположим теперь, что $ L_{x} $ и $ L_{x}^{\ast} $ конечномерные пространства одной и той же размерности $ n $. Выберем в $ L_{x} $ и $ L_{x}^{\ast} $ какие нибудь бзисы; обозначим их соответственно через $ \bar{e}_{1},..., \bar{e}_{n} $ и $ \tilde{e}_{1},..., \tilde{e}_{n} $. Для произвольных элементов $ \omega \in L_{x}^{\ast} $, и $ \xi \in L_{x}$ напишем разложения

\beq \label{razl_po_baz}
\omega = a_{1} \tilde{e}^{1} + ... + a_{n} \tilde{e}^{n}, \\
\xi = \xi^{1} \bar{e}_{1} + ... \xi^{n} \bar{e}_{n}.
\eeq

Вследствие (\ref{razl_po_baz} ) имеем следующее общее выражение свертки:

\beq \label{svertka}
( \omega, \xi ) = \sum ( \tilde{e}^{i}, \bar{e}_{j} )\omega_{i} \xi^{j}
\eeq

Из ( \ref{svertka} ) видно, что свертка будет определена на
$ L_{x}^{\ast} \times L_{x} $, если мы зададим матрицу сверток базисных элементов, т.е. матрицу чисел $ ( \tilde{e}^{i}, \bar{e}_{j} ) $. Легко усмотреть, что для обеспечения обоих условий невыроженности 3) и 4) необходимо и достаточно, чтобы эта матрица была невыроженной; таким образом,

\beq \label{det_ee}
\mathrm{det} ( \tilde{e}^{i}, \bar{e}_{j} ) \neq 0
\eeq

В некоторых специальных базисах $ \tilde{e}^{i} $, $ \bar{e}_{j} $ матрицу $ ( \tilde{e}^{i}, \bar{e}_{j} ) $ можно сделать единичной. Вместе с тем упростится выражение (2). Именно, имеет место
\textit{Теорема.} Пусть на $ L_{x}^{\ast} \times L_{x} $ как угодно задана свертка $ ( \omega, \xi )  $ и в $ L_{x}^{\ast} $ как угодно задан базис $ \tilde{e}^{1},...,\tilde{e}^{n} $; тогда в $ L_{x} $ найдется единственный базис $ \bar{e}_{1},...,\bar{e}_{n} $ такой, что

\beq \label{vzaimnost}
(\tilde{e}^{i}, \bar{e}_{j}) = \delta_{j}^{i}
\eeq

где $ \delta_{j}^{i} $ - символ Кронекера. Роли $ L_{x}^{\ast} $ и $ L_{x} $ можно обменять.

\textit{Доказательство.} Для любого набора чисел $ a^{1},...,a^{n} $ найдется единственный вектор $ \zeta \in L_{x} $ такой, что $ ( \tilde{e}^{1}, \zeta ) = a^{1} ,..., ( \tilde{e}^{n}, \zeta ) = a^{n} $. Чтобы убедится в этом разложим искомый вектор $ \zeta $ по какому-нибудь базису:
$ \zeta = \lambda^{1}\bar{e}_{1} + ...+ \lambda^{n}\bar{e}_{n} $. Мы получим для $ \lambda^{1},..., \lambda^{n} $ систему уравнений первой степени с главной матрицей $ ( \tilde{e}^{i}, \bar{e}_{j} ) $; полученная система однозначно рзрешим в следствие ( \ref{det_ee} ).

Беря теперь в качестве $ a^{1}, a^{2},..., a^{n} $ набор чисел $ 1, 0,..., 0 $ найдем по предидущему вектор $ \zeta $. Положи $ \bar{e}_{1} = \zeta $. Аналогично по набору
$ 0,1,0,...,0 $ найдем $ \bar{e}_{2} $ и т.д. Полученные векторы $ \bar{e}_{1}, \bar{e}_{2},...,\bar{e}_{n} $ удовлетворяют равенствам ( \ref{vzaimnost} ). Из этих же равенств следует, что векторы $ \bar{e}_{1}, \bar{e}_{2},..., \bar{e}_{n} $ линейно независимы.

\textit{Определение.} Два базиса, из которых один принадлежит пространству $ L_{x}^{\ast} $, другой - пространству $L_{x} $, навываются \textit{взаимными} или \textit{дуальными}, если они удовлетворяют равенствам
( \ref{vzaimnost} ). Соответственно имеем

\beq \label{dualniy}
(\tilde{e}^{i}, \bar{e}_{j}) = \delta_{j}^{i}, \hspace{1ex} \text{где} \tilde{e}^{i} \in L_{x}^{\ast}, \bar{e}_{j} \in L_{x}.
\eeq

Теперь, если разложения ( \ref{razl_po_baz} ) даны по  взаимным базисам, то
\beq \label{scalar_product}
(\omega, \xi) = \omega_{1}\xi^{1}+ ... + \omega_{n}\xi^{n}
\eeq


Это следует прямо из ( \ref{svertka} ) и ( \ref{dualniy} ).

Из ( \ref{det_ee} ) следует, что разменрость $ L_{x}^{\ast}  $ равна размерности $ L_{x} $, т.е $ n $.

Назначим свертку двух произвольных элементов $ \omega \in L_{x}^{\ast} $ и $ \xi in L_{x} $, полагая

\beq \label{chislo_znach_formi}
( \omega, \xi ) = \omega ( \xi )
\eeq

т.е. в качестве $ (\omega, \xi ) $ мы берем сейчас число, равное значению формы $ \omega \in L_{x}^{\ast} $ на элементе $ \xi L_{x} $. Требования которые предъявляются к свертке соблюдены.

Из теоремы (п3) и из ( \ref{chislo_znach_formi} ) следует, что каковы бы ни были линейно-независимые формы $ \tilde{e}^{1} ( \xi ),..., \tilde{e}^{n}( \xi ) $, $ \xi \in L_{x}  $, в пространстве $ L_{x} $ найдется единственный базис $ \bar{e}_{1},...,\bar{e}_{n} $ такой, что

\beq \label{e(e)=delta}
\tilde{e}^{i} ( \bar{e}_j ) = \delta_{j}^{i}
\eeq

Справедливо также утверждение, что для любого бзиса $ \bar{e}_{1},...,\bar{e}_{n} $ в
$ L_{x}  $ найдется единственная линейно-независимая система форм
$ \tilde{e}^{1}( \xi ),.., \tilde{e}^{n}(\xi) $, подчиненная условиям (10).
%шутц стр 82.
Чтобы убедится в этом покажем, что 1-формы $ \tilde{e}^{i} $ линейно-независимы и, значит образуют базис. Для этого рассмотрим любую 1-форму $ \theta $, действующую на произвольный вектор $ \zeta$
\beq \label{forma_na_pr_vector}
\theta ( \zeta ) = \theta ( \sum_{j} \zeta^{j}\bar{e}_{j} )=
\sum_{j} \zeta^{j} \theta( \bar{e}_{j} )= \sum_{j} \tilde{e}^{j} ( \zeta ) \theta ( \bar{e}_{j} ) 
\eeq

Величины $ \theta_{j} = \theta ( \bar{e}_{j} ) $ \textit{называтются компонентами} $ \theta $ в базисе, дуальном к $ \lbrace \bar{e}_{j} \rbrace $.
Перепишем (\ref{forma_na_pr_vector}) в виде
\beq
\theta ( \zeta ) = \sum \theta_{j} \tilde{e}^{j} ( \zeta ) 
\eeq

Поскольку 1-форма \textit{определяется} своими значениями на векторах, из этого равенства вместе с (2.16) вытекает, ввиду произвольности $ \zeta $, что
\beq \label{forma_po_bazisu}
\theta = \sum_{j}\theta_{j} \tilde{e}^{j}
\eeq

Отсюда следует, что $ \lbrace \tilde{e}^{j} \rbrace $ действительно образуют базис, поскольку их как раз $ т $ штук и любоая 1-форма $ \theta $ представима в виде их линейной комбинации. Отсюда следует также, что величины $ \lbrace \theta_{i} \rbrace $ на самом деле являются компоненатами $ \theta $ относительно базиса в обыном смысле слова.

Важно, что у нас есть теперь формула для вычисления значения $ \theta ( \zeta ) $ по известным компонентам $ \theta $ и
$ \zeta $:
\beq \label{forma_po_componontam}
\theta (\zeta) = \sum_{j} \theta_{j} \zeta^{•}
\eeq

Как отмечалось выше, это и есть свертка $ \zeta $ и $ \theta $.

Пусть произвольный вектор $ \zeta $ разложен по базису $ \lbrace \bar{e}_{i} \rbrace $

\beq
\zeta = \zeta^{1}\bar{e}_{1} + ... + \zeta^{n}\bar{e}_{n} \nonumber
\eeq

Тогда
\beq
\tilde{e}^{j}( \zeta ) = \zeta^{1}\tilde{e}^{j}(e_{1})+...+\zeta^{j}\tilde{e}^{j}(e_{j})+...+\zeta^{n}\tilde{e}^{j}(e_{n})
\eeq
Отсюда и в следствие ( \ref{e(e)=delta} ) имеем
\beq
\tilde{e}^{j} ( \zeta ) = \zeta^{j}
\eeq

Это значит, что координатная запись формы $ \tilde{e}^{j} $ при употреблении взаимного с ними базиса оказывается особенно простой: все коэффициенты форм $ \tilde{e}^{j} $ равны нулю, крое одного, который занимает $j$-е место и равен единице.

Вместе с тем можно сказать, что координаты любого вектора по базису $ \lbrace \bar{e}_{i} \rbrace $ суть значения на этом векторе форм взаимного базиса, что выражается равенством
\beq
\zeta = \tilde{e}^{1}(\zeta)\bar{e}_{1} +...+ \tilde{e}^{n} (\zeta) e_{n}
\eeq

\section{Полилинейные формы. }

Обозначим через $ (L_{x})^{k} $ декартово произведение пространства $ L_{x} $ на себя:
$ (L_{x})^{k} = L_{x} \times L_{x} \times ... \times L_{x} $ ($ k $-раз). По определению пространства $ (L_{x})^{k} $ его элементами являются упорядоченные наборы векторов из $ L_{x} $, взятых в числе $ k $; таким образом, $ (\xi_{1},...,\xi_{n} \in (L_{x})^{k} $, если каждый из векторов $ \xi_{1}..., \xi_{n} $ принадлежит $ L_{x} $.

\textit{Полилинейную форму} от векторынх аргументво определим как действительную функцию $ \Omega $ в $ (L_{x})^{k} $ при условии ленейности по каждому (векторному) аргументу
\beq
\Theta(\alpha \xi_{1}^{'} + \beta \xi_{1}^{''}, \xi_{2},..., \xi_{k}) = \\
\alpha \Theta ( \xi_{1}^{'}, \xi_{2},..., \xi_{k} )+
\beta \Theta (\xi_{1}^{''}, \xi_{2},..., \xi_{k}).
\eeq

Здесь условие линейности записано для первого аргумента. Полилинейную форму мы обозначили также как и 1-форму (линенйную) так как 1-форма частный случай полилинейной формы.

Рассмотрим произвольную полилинейную форму $ \Theta $. Возьмем любые линейные формы $ \tilde{e}^{1},...,\tilde{e}^{n} $ в числе $ n $ при единственном услови их линенйной независимотсти.

\textit{Теорема.} Численное значние $ \Theta(\xi^{1},..., \xi^{2} ) $ формы $ \Theta $ может быть предствавленно в виде
\beq \label{chis_znach_formi}
\Theta (\xi_{1},...,\xi_{n}) =
\sum \Theta_{j_{1} ... j_{k}}\tilde{e}^{j_{1}}(\xi_{1}) ...
\tilde{e}^{j_{k}}(\xi_{k})
\eeq
где $ \Theta_{j_{1} ... j_{k}} $ - коэффициенты, которые определяются данной формой $ \Theta $, а также выбором системы линейно-независимых форм $ \lbrace \tilde{e}^{n} \rbrace $.

\textit{Доказательство.} Пусть $ \bar{e}_{1},..., \bar{e}_{n} $ - базис в $ L_{x} $, взаимный с базисом $ \lbrace \tilde{e}^{n} \rbrace $ в $ L_{x}^{\ast} $. Запишем каждый из векторов $ \xi_{1},..., \xi_{k} $ в виде:
\beq
\xi_{1} = \tilde{e}^{1}(\xi_{1})\,\bar{e}_{1} + 
\tilde{e}^{n}(\xi_{1})\,\bar{e}_{n} \\
...\\
\xi_{k} = \tilde{e}^{1}(\xi_{k})\,\bar{e}_{1} + 
\tilde{e}^{n}(\xi_{k})\,\bar{e}_{n}
\eeq
Подставим эти разложения в левую часть ( \ref{chis_znach_formi} ) и воспользуемся линейностью формы $  $ по каждому аргументу. Мы получим правую часть выражения ( \ref{chis_znach_formi} ), где положено
\beq \label{koef_formi}
\Theta_{j_{1} ... j_{k}} = \Theta( \bar{e}_{j_{1}},..., \bar{e}_{j_{k}}  )
\eeq
Теорема доказана.

Выражение \ref{chis_znach_formi} само по себе, т.е. будучи уже доказанным, не предусматривает использование какого-либо базиса. Если же пользоватся базисом $ \lbrace e_{n} \rbrace $, который употреблялся в доказательстве, то формулы $ \lbrace \tilde{e}^{1}(\xi) \rbrace $ можно выразить в нем известным нам специальным образом; именно,
\beq
\tilde{e}^{j}(\xi_{k}) = \xi^{j}_{k},
\eeq

где справа написана $j$-я координата вектора $ \xi_{k} $ в базисе $ \lbrace \bar{e}_{n} \rbrace $, В этом базисе выражение (\ref{chis_znach_formi} ) принимает вид
\beq \label{chis_znach_formi_2}
\Theta (\xi_{1},...,\xi_{n}) =
\sum \Theta_{j_{1} ... j_{k}} \xi_{1}^{j_{1}} ... \xi_{k}^{j_{k}}
\eeq
Мы получаем представление формы $ \Theta (\xi_{1},...,\xi_{n}) $ в базисе $ \lbrace \bar{e}_{n} \rbrace $. Этот базис в $ L_{x} $ можно взять заранее и вполне произвольно. Взаимный с ним базис найдется, но при употреблении запии () может не использоватся.

Из \ref{koef_formi} следует, что для даннй формы $ \Theta $ и для данных базисных форм $ \lbrace \tilde{e}^{1}(\xi) \rbrace $, разложение \ref{chis_znach_formi} единственно, т.е. численные значения коэффициентов $ \Theta_{j_{1} ... j_{k}} $ однозначно опредлены индексами $ j_{1} ... j_{k} $.\\

\section{Пространство полилинейных форм. }

Полилинейные формы, определенные в пространстве $ (L_{x})^{k} $, образуют линейное пространство, елсли для произвольных двух таких форм $ \Theta $ и $ \Omega $ определить линейные операции согласно равенству
\beq
( \alpha \Theta + \beta \Omega ) ( \xi_{1},..., \xi_{k} ) =
\alpha \Theta (\xi_{1},...,\xi_{k}) + \beta \Omega (\xi_{1},...,\xi_{k}).
\eeq

Это линейное пространство мы обозначим через 
$ (T_{x})^{k}[L_{x}] $ или просто через $ (T_{x})^{k} $ и будем называть $k$-кратным тензорным произведением сопряженного пространства
$ L_{x}^{\ast} $ на себя. Символически
\beq
(T_{x})^{k} = (T_{x})^{k}[L_{x}] = T_{x}^{\ast} \otimes T_{x}^{\ast} ... T_{x}^{\ast}.
\eeq
Заметим, что само пространство $ L_{x}^{\ast} $ есть $ (T_{x})^{1} $. Поэтому
\beq
(T_{x})^{k} = (T_{x})^{1} \otimes (T_{x})^{1} ... (T_{x})^{1}
\eeq

Полилинейные формы как элементы пространства $ (T_{x})^{k} $ называются $k$-тензорами, подробнее, - \textit{ковариантными тензорами} велентности $k$ в пространстве $L_{x}$. Линейные формы как элементы пространства $ (T_{x})^{1} = L_{x}^{\ast} $ называются одновалентными ковариантными тензорами, или \textit{ковариантрынми векторами}, или \textit{ковекторами}. Действительные числа мы будем называть \textit{тензорами нулевой валентности.}

Поменяв ролями $ L_{x} $ и $ L_{x}^{\ast} $, получим аналогично предидущему \textit{контравариантные тензоры валентности} $ k $, как полилинейные формы, отображающие
$ ( L_{x}^{\ast} )^{k} $ в $ \mathbb{R} $.

\textit{Определение.} Тензорным произведением или просто произведением тензора $ \Theta $ на тензор $ \Omega $
\beq
\Theta \in (T_{x})^{k}[L_{x}], \Omega \in (T_{x})^{k}[L_{x}]
\eeq
называется тензор, который обозначается $ \Theta \otimes \Omega $, принадлежит пространству
$ (T_{x})^{k+l}[L_{x}] $ и определеяется в виде полилинейной формы равенством
\beq \label{tens_times}
(\Theta \otimes \Omega )( \xi_{1},..., \xi_{k}, \xi_{k+1},.., \xi_{k+l} ) = \\
\Theta(\xi_{1},...,\xi_{k})\Omega( \xi_{k+1},..., \xi_{k+l})
\eeq


Здесь слева и справа написаны чиселнные значения полилинейных фомр при любом выборе векторов
$ \xi_{1},...,\xi_{k},...,\xi_{k+1},...,\xi_{k+l} \in L_{x}$, взятых независимо друг от друга.

Тензорное произведение обладает следующими свойствами:
\beq
( \Theta_{1} + \Theta_{2} ) \otimes \Omega = 
\Theta_{1} \otimes \Omega + \Theta_{2} \otimes \Omega,\\
\Theta \otimes ( \Omega_{1} + \Omega_{2} ) = \Theta \otimes \Omega_{1} + \Theta \otimes \Omega_{2},\\
( \alpha \Theta ) \otimes \Omega = \alpha( \Theta \otimes \Omega  ),\\
( \Theta \otimes \Omega ) \otimes \Phi =
\Theta \otimes ( \Omega \otimes \Phi ).
\eeq

Последнее свойств позволяет писать $ \Theta \otimes \Omega \otimes \Phi $ без указания ассоциаций. Вместес тем определено тензорное произведение любого числа тензорных сомножителей.

Вообще говоря $ \Theta \otimes \Omega $ не совпадает с $ \Omega \otimes \Theta $ (достаточно написать
( \ref{tens_times} ) для $ \Omega \otimes \Theta $.
\textit{Пример.} У функции $ f(x,y) = 3x + 5y $  значнения
$ f(1,2) $ и $ f(2,1) $ различны.

Равенство тензоров $ \Theta $ и $ \Omega $ следует понимать как тождественное совпадение полилинейных форм $ \Theta:\, (L_{x})^{k} \longrightarrow \mathbb{R} $ и $\Omega:\, (L_{x})^{k} \longrightarrow \mathbb{R}  $. Равенство $ \Theta $ означает отображение $ \Theta:\, (L_{x})^{k } \longrightarrow 0 $.

Пусть $ \tilde{e}^{1},...,\tilde{e}^{n} $, линейно-независимые линейные формы в $ L_{x} $. Мы рассматриваем их как одновелентные тензоры, т.е. как элементы пространства
$ (T_{x})^{1} = L_{x}^{\ast} $. Тогда
\beq \label{tens_basis}
\tilde{e}^{j_{1}} \otimes \tilde{e}^{j_{2}} \otimes ... \otimes \tilde{e}^{j_{k}} \in (T_{x})^{k}
\eeq

\textit{Теорема.} Множество всех тензорных произведений ( \ref{tens_basis} ), т.е отвечающих всевозможным наборам индексов $ j_{1},...,j_{k};\, (j_{1} = 1,2,...,n ...; j_{k} = 1,2,...,n) $ составляет базис в $ (T_{x})^{k} $.

\textit{Доказательство.} Доказательство этой теоремы еквивалентно доказательству выражения (\ref{chis_znach_formi}). Установленные там предложения означают, что для любого $ \Theta \in (T_{x})^{k} $ имеет место единственное разложение
\beq \label{razlozh_po_tens_bisis}
\Omega = \sum \Omega_{j_{1}...j_{k}} \tilde{e}^{j_{1}} \otimes ... \otimes \tilde{e}^{j_{k}}
\eeq
А это и значит (см.формулу (\ref{chis_znach_formi})), что
$ \tilde{e}^{j_{1}} \otimes ... \otimes \tilde{e}^{j_{k}} $ составляют базис.

\textit{Следствие.} Размерность пространства $ (T_{x})^{k} $ равна $ n^{k} $.
\textit{Замечание.} Коэффициенты $ \Theta_{j_{1}...j_{k}} $ разложеня ( \ref{razlozh_po_tens_bisis} ) называются координатами тензора $ \Theta $ (по базису $ \tilde{e}^{j_{1}} \otimes ... \otimes \tilde{e}^{j_{k}} $ в
$ (T_{x})^{k} $ ).

Итак, мы имеем счетную последовательность пространств
\beq
(T_{x})^{0}, (T_{x})^{1}, (T_{x})^{2},..., (T_{x})^{k},...,
\eeq
построенных по данному линейному пространству $ L_{x} $. В каждом $ (T_{x})^{k} $ определены линейные операции, в каждой паре $ (T_{x})^{k},\, T_{x}^{l} $ (допуская $ k=l $) определена операция тензорного произведения:
\beq
\Theta \otimes \Omega, \Theta \in (T_{x})^{k},\, \Omega \in (T_{x})^{k},\, \Theta \otimes \Omega \in (T_{x})^{k+l}. 
\eeq
Элементы этих пространств, как объекты указнных операций, называются
\textit{ковариантными тензорами} (над $ L_{x} $). Если $ \Theta \in (T_{x})^{k} $, то $ \Theta $ называется \textit{Тензором валентности $ k $}. Пространство $ (T_{x})^{k} $ является $ n^{k} $-мерным.\\

\textit{\textbf{$ k' $-раз контравариантные и $ k $-раз коварантные тензоры.}}

%Шутц
Рассматривая полилинейную форму формально, мы можем определить ее как функцию от $ k' $ векторов и $ k $ ковекторов.

\textit{Определение.} Линейная функция, агрументами которой служат $ k $ один форм и $ k' $ векторов, а значениями - вещественные числа танывается \textit{тензором типа 
$ \left(^{k}_{k'} \right) $.} Это определение является обобщением данного выше для полилинейных форм.

Например, если $ F $ - тензор типа $ \left(^{2}_{2} \right) $, то его значение на 1-формах $ \omega $ и $ \sigma $ и векторах $ \xi $ и $ \zeta $ записывается в виде
\beq \label{kov_kont_tens}
F(\omega, \sigma; \xi, \zeta ).
\eeq

Свойство линейности здесь означает, что (для произвольных чисел $ \alpha $ и $ \beta $)
\beq \label{lin_kk'_tensor}
F( \alpha \omega_{1} + \beta \omega_{2}, \sigma; \xi, \zeta ) =\\
=\alpha F( \omega_{1}, \sigma; \xi, \zeta ) +
\beta F( \omega_{2}, \sigma; \xi, \zeta )
\eeq
и аналогично для других элемнтов. В случае если мы хотим иметь дело с $ F $ не указывая явно аргументы, мы будем использовать обозначение $ F(\, , \, ;\, ,\,)$, где на пустые места можно поставить любые агрументы соответствующего типа, но нужно помнить, что порядок их сделования вообще говоря существеннен.

\textit{Определение.} Тензорное поле типа $ \left(^{k}_{k'} \right) $ - это правило, сопоставляющее каждой точке тензор типа $ \left(^{k}_{k'} \right) $ в этой точке. Свойство линейности тензоров распространяется и на тензорные поля, причем числа $ \alpha $ и $ \beta $ в ( \ref{lin_kk'_tensor} ) в разных точках могуть быть различными  - они являются функциями на $ M $.

Отметим, что векторы суть тензоры типа $ \left(^{1}_{0} \right) $: они являются линейными функциями на 1-формах. Аналогично 1-формы суть тензоры тпиа $ \left(^{0}_{1} \right) $. Скалярные функции принято считать тензорами типа
$ \left(^{0}_{0} \right) $.

Тензор $ F $ типа $ \left(^{1}_{1} \right) $ предполагает наличие двух аргументов. Значение $ F ( \omega ; \xi) $ есть вещественное число. При фиксированном
$ \omega $ мы получаем 1-форму $ F(\omega; \, ) $, поскольку сюда можно подставить вектор, для того, чтобы получить вещественное число. При фиксиованном $ \omega $ мы получаем вектор, для того чтобы получить вещественное число сюда нужно подставить 1-форму. Таким образом, тензор типа $ \left(^{1}_{1} \right) $ можно рассматривать как линейную функцию на векторах со значениями в векторах, а также как линейную функию на 1-формах, принимающую значения также в 1-формах.

В соответствии в установленными обозначениями выражене (\ref{scalar_product}) является тензором типа $ \left(^{1}_{1} \right) $ а его компоненты записываются в виде $ \omega_{j} \xi^{i} \rbrace $.

%щутц. свертка.
Итак, суммируя диагональные компоненты $ ( i = j ) $, получим $ \omega_{j} \xi^{i} $ - число, не зависящее от базиса, а именно значение $ \omega $ на $ \xi $, которое можно считать тензором типа $ \left(^{0}_{0} \right) $. Аналогично, если $ S^{i}_{jk} $ и $ P^{lm} $ компоненты тензоров типа $ \left(^{1}_{2} \right) $ и $ \left(^{3}_{2} \right) $ соответсвтенно, то $ S^{i}_{jk} P^{lm} $ суть компоненты тензора типа $ \left(^{3}_{2} \right) $, $ S^{i}_{jk} P^{jm} $ - компоненты тензора типа $ \left(^{2}_{1} \right) $, $ S^{i}_{jk} P^{lj}  $ - компоненты другого тензора типа $ \left(^{2}_{1} \right) $ и т.д. По аналогии с формулой (\ref{svertka}) эта операция называется всерткой; она позволяет получать новые тензоры из уже имеющихся.

Дадим доказательство независимости свертки от выбора базиса. Рассмотрим тензор $ A $ типа $ \left(^{2}_{0} \right) $, тензор $ B $ типа $ \left(^{0}_{2} \right) $ и их свертку (относительно некоторого базиса) $ A^{ij}B_{jk} $. Мы утверждаем, что это - компоненты тензора $ C $ типа $ \left(^{1}_{1} \right) $, такого что для произвольных вектора $ \xi $ и 1-формы $ \sigma $
\beq \label{tensor_C}
C( \sigma; \xi ) \ \left( \sum_{j} A^{ij}B_{jk} \right)\sigma_{i} \xi^{k} =\\
= \sum A( \sigma, \omega^{j} ) B ( \bar{e}_{j}, \xi ).
\eeq

В силу линейности $ A $ по второму грументу это можно переписать в виде
\beq
C( \sigma; \xi ) = A \left( \sigma, \sum_{j}B( \bar{e}_{j}, \xi ) \omega^{j} \right),
\eeq
так как величины $ B( \bar{e}_{j}, \xi ) $ суть просто числа. Но в силу независимости светки от выбора базиса
\beq
\sum_{j} ( \bar{e}_{j}, \xi )\omega^{j} = B(\, ,\xi)
\eeq
где правая чатсь является один-формой (при фиксированном
$ \xi $) - на пустое место нужно подставлять вектор. Эта один-фомра стоит в качестве одного из аргументов в $ A $, так что
\beq
A^{ij}B_{jk} = C^{i}_{k} \Leftrightarrow C( \sigma; \xi ) =
A( \sigma, B(\, ,\xi ) ),
\eeq
независимо от выбора базиса.
